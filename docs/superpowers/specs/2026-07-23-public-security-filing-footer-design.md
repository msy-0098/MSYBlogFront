# 公安备案信息页脚展示设计

- 日期：2026-07-23
- 范围：前端公共页脚
- 状态：已获用户确认

## 1. 背景

博客域名 `masenyu.top` 已完成公安联网备案，备案号为 `冀公网安备13060602001918号`。公共页脚当前已经展示工信部 ICP 备案号，需要继续补充公安备案标准图标、备案号与官方查询链接。

## 2. 目标

1. 在公共页脚展示公安备案号 `冀公网安备13060602001918号`。
2. 在备案号左侧展示公安备案图标，并将图标作为前端本地静态资源发布，避免依赖外站图片。
3. 图标与备案号组成同一个可点击链接，跳转公安部备案查询详情页。
4. 保持现有 ICP 备案链接、RSS、Sitemap 和技术栈信息不变。
5. 在桌面端保持紧凑排列，在窄屏下允许自然换行。
6. 保持后台管理路由不显示公共页脚的现有行为。

## 3. 非目标

- 不修改后端接口、数据库或站点配置。
- 不改变 ICP 备案号或其工信部查询链接。
- 不调整公共页脚的整体视觉体系。
- 不使用远程图片 URL，不引入第三方前端依赖。

## 4. 方案

### 4.1 备案链接

在 `src/components/layout/AppFooter.vue` 的现有页脚链接区域中加入公安备案链接：

- 文案：`冀公网安备13060602001918号`
- 链接：`https://beian.mps.gov.cn/#/query/webSearch?code=13060602001918`
- 属性：`target="_blank"`、`rel="noreferrer"`
- 测试定位：`data-test="public-security-filing-link"`

### 4.2 图标资源

新增本地图标资源 `public/images/beian-police.png`：

- 使用公安备案常见盾牌图标；
- 保持原始宽高比；
- 页面显示尺寸约 `18px × 18px`；
- 图片设置空 `alt` 并配合 `aria-hidden="true"`，避免屏幕阅读器重复朗读，备案链接本身提供完整文本名称。

### 4.3 布局与样式

- 公安备案链接使用 `inline-flex`，图标和文字垂直居中；
- 图标与备案号之间保留约 `4px` 间距；
- 链接颜色与现有页脚链接一致；
- `.footer-links` 继续使用现有 flex 与换行能力，移动端不强制挤在同一行。

## 5. 测试与验收

1. 组件测试验证备案号、公安部查询链接、外链安全属性和本地图标路径。
2. 运行 `pnpm test`。
3. 运行 `pnpm build`。
4. 运行 `git diff --check`。
5. 启动页面并进行桌面端、移动端视觉核验，确认图标清晰、文字对齐且无溢出。
6. 部署前端后访问生产站点确认公安备案信息可见、可点击。
7. 提交前执行 `git status --short`，排除依赖、构建产物、缓存、日志和敏感配置；前端仓库分别推送 GitHub `origin` 与 Gitee `gitee`。
